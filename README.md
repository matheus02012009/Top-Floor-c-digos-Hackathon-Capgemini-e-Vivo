# Top-Floor-c-digos-Hackathon-Capgemini-e-Vivo
código utilizado no hackathon
💡 LuzGuia

O LuzGuia é uma aplicação web que usa inteligência artificial para reconhecer produtos em uma prateleira através da câmera e sugerir itens relacionados com base na preferência do cliente.

🚀 Como funciona
O usuário escolhe:
Marca preferida (Samsung, Apple ou outro)
Se quer produtos exclusivos
A câmera é ativada e a IA identifica o produto na prateleira
O sistema analisa:
O que foi detectado pela câmera
As preferências do usuário
Um produto relacionado é exibido na tela
🧠 Tecnologias usadas
TensorFlow.js
Teachable Machine
HTML, CSS e JavaScript
⚙️ Como usar
Abra o arquivo index.html no navegador
Permita o uso da câmera
Escolha suas preferências
Aponte a câmera para a prateleira
📌 Objetivo

O projeto demonstra como usar inteligência artificial para criar uma prateleira inteligente, melhorando a experiência do cliente com recomendações automáticas.
